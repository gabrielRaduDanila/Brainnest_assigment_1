1. What do HTML and CSS stand for?

    2. HTML (the Hypertext Markup Language) and CSS (Cascading Style Sheets) are two of the core technologies for building Web pages. HTML provides the structure of the page, CSS the (visual and aural) layout, for a variety of devices. Along with graphics and scripting, HTML and CSS are the basis of building Web pages and Web Applications.

  2. Would you rather use HTML or CSS, for putting paragraphs of text on a webpage?

    3. For putting paragraphs of text on a webpage we will use HTML.

  3. Would you rather use HTML or CSS, to change the font and background color of a button?

    4. We will use CSS for styling a button (background and font size)

  4. What is an HTML tag?

    5. The HTML tag tells the browser when it starts/closes an HTML element and what kind of element.

  5. What are the three parts of an HTML element? (image from paragraph)

    6. The three parts of an HTML tag are: the opening tag, content, and closing tag

  6. What is the purpose of the doctype declaration?

    7. The purpose of the doctype declaration is to tell the browser what version of HTML it should use to render the document.

  7. What is the HTML element?

    8. HTML element it is known as the root element of the document

  8. What is the purpose of the head element?

    9. The head element is where we put important meta-information about our webpages.

  9. What is the purpose of the body element?

    10. In the body element Is where all the content that will be displayed to users will go - the text, images, lists, links, and so on

  10. How do you create a paragraph in HTML?

    11. We create it with the p tag

  11. How do you create a heading in HTML?

    12. We create it with the h tag

  12. How many different levels of headings are there and what is the difference between them?

    13. There are 6 levels of headings. h1 headings should be used for main headings, followed by h2 headings, then the less important h3, and so on.

  13. What element should you use to make text bold and important?

    14. We should use the strong element.

  14. What element should you use to make text italicized to add emphasis to it?

    15. We should use the em element.

  15. What relationship does an element have with any nested elements within it?

    16. Nested elements are children of their parent container.

  16. What relationship do two elements have if they are at the same level of nesting?

    17. Many child elements can be placed into a parent element making a sibling relationship among the group of nested children.

  17. How do you create HTML comments?

    18. We will write < !-- comment -->

  18. What HTML tag is used to create an unordered list?

    19. The ul tag is used

  19. What HTML tag is used to create an ordered list?

    20. The ol tag is used

  20. What HTML tag is used to create list items within both unordered and ordered lists?

    21. The li tag is used

  21. What element is used to create a link?

    22. The `a` element (or anchor element)

  22. What is an attribute?

    23. Attributes provide additional information about elements.

  23. What attribute tells links where to go?

    24. The href attribute tells links where to go.

  24. What is the difference between an absolute and relative link?

    25. An absolute link provides the complete location information of your site, while a relative link only contains the location following your domain.

  25. Which element is used to display an image?

    26. The img element is used to display an image.

  26. What two attributes do images always need to have?

    27. The img element has two required attributes: src : The source location (URL) of the image file. alt : The alternate text.

  27. How do you access a parent directory in a file path?

    28. We access a parent directory by putting ../ in the href attribute.

  28. What are the four main image formats that you can use for images on the web?

    29. JPEG, PNG, SVG and GIF are the four main image formats that you can use for images on the web.
